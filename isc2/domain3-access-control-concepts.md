# ISC2 CC Domain 3 - Access Control Concepts

## Status

Completed ✅

## Overview

Domain 3 focuses on controlling who can access systems, data, and resources.

Access control is one of the most important security concepts because it helps protect information from unauthorized access.

## Identification, Authentication, and Authorization

### Identification

A user claims an identity.

Examples:

* Username
* Email address
* Employee ID

### Authentication

The system verifies the user's identity.

Examples:

* Passwords
* Biometrics
* Security tokens

### Authorization

Determines what an authenticated user is allowed to do.

Examples:

* Read files
* Modify records
* Access applications

## Principle of Least Privilege

Users should only receive the minimum permissions necessary to perform their job functions.

Benefits:

* Reduces risk
* Limits damage from compromised accounts
* Improves security management

## Access Control Models

### Role-Based Access Control (RBAC)

Permissions are assigned based on job roles.

Example:

* Administrator
* Manager
* Employee

### Mandatory Access Control (MAC)

Access is controlled by security classifications and policies.

### Discretionary Access Control (DAC)

Resource owners decide who receives access.

## Multi-Factor Authentication (MFA)

MFA requires two or more forms of verification.

Examples:

* Password + One-Time Code
* Password + Fingerprint

Benefits:

* Improves account security
* Reduces risk of unauthorized access

## Monitoring and Surveillance

Organizations use monitoring systems to detect and respond to security events.

Examples:

* CCTV cameras
* Security guards
* Access logs
* Alarm systems

## Physical Access Controls

Physical controls help prevent unauthorized physical access.

Examples:

* Locks
* Security badges
* Fences
* CCTV systems

## Key Takeaways

* Access control protects information and resources.
* Authentication verifies identity.
* Authorization determines permissions.
* MFA strengthens security.
* Physical and logical controls work together to protect assets.
